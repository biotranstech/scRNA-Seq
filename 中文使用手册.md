scRNA-Seq RNA Editing Workflow 使用指南
======

## 项目简介（Overview）
本项目是一个基于 Python DAGflow 构建的：

👉 单细胞 RNA 编辑（scRNA RNA editing）自动化分析流程

可以完成：

1. 从原始 FASTQ 数据质控 
2. 单细胞分析
3. RNA 编辑检测
4. 统计分析与可视化

## 为什么要使用这个流程
单细胞 RNA 编辑分析通常非常复杂：
```
| 问题   | 传统方法                    |
| ---- | ----------------------- |
| 工具多  | cellranger + 自写脚本 + R分析 |
| 流程复杂 | 多阶段手动执行                 |
| 易出错  | 路径、格式、参数                |
| 不可复现 | 每个人流程不同                 |
```
✅ 本流程优势

👉 一键完成：
✔ 单细胞处理（cellranger）
✔ RNA editing 分析
✔ 自动统计
✔ 自动分析

👉 适用于：
1. 单细胞RNA编辑研究
2. 肿瘤微环境分析
3. RNA调控机制研究

## 流程逻辑（Workflow）
```
FASTQ
 ↓
Cell Ranger（单细胞分析）
 ↓
表达矩阵生成
 ↓
RNA editing检测
 ↓
统计分析
 ↓
结果输出
```

## 项目结构

```
├── up_analysis.py        # 第一阶段（scRNA分析）
├── after_anno.py        # 第二阶段（RNA editing分析）
├── config.py            # 软件路径配置（必须修改）
├── common.py            # 通用模块
├── dagflow/             # DAGflow框架
├── scripts/             # 分析脚本
├── sample.list          # 样本信息文件
└── human_common_marker.xlsx  # marker基因信息
```

## 安装步骤

### 安装 Conda（如果没有）
推荐：
```
Miniconda
Anaconda
```

### 安装 Python（推荐3.8+）
```
conda create -n scrna python=3.9 -y
conda activate scrna
```

### 安装 R 环境（关键）
创建 environment.yml：
```
name: scrna_r
channels:
  - conda-forge
  - bioconda
dependencies:
  - r-base=4.2.3
  - r-seurat
  - r-tidyverse
  - r-data.table
  - r-openxlsx
  - r-optparse
  - bioconductor-annotationdbi
```
运行：
```
conda env create -f environment.yml
conda activate scrna_r
```

### 安装核心软件
必须安装以下工具：
10X官方单细胞分析软件：cellranger（https://support.10xgenomics.com/single-cell-gene-expression/software/pipelines/latest/what-is-cell-ranger）
统计分析以及可视化分析软件：R (https://www.r-project.org)

### 配置（必须修改）
编辑：
```
config.py
```
修改：
1. cellranger 路径
2. Rscript 路径
3. 参考基因组路径
👉 不改 = 100%跑不通

### 输入文件准备
sample.xls（核心输入）:
```
sample1   /path/sample1_R1.fastq.gz   /path/sample1_R2.fastq.gz   group1
sample2   /path/sample2_R1.fastq.gz   /path/sample2_R2.fastq.gz   group1
sample3   /path/sample3_R1.fastq.gz   /path/sample3_R2.fastq.gz   group2
```
📌 格式说明
```
| 列   | 内容   |
| --- | ---- |
| 第1列 | 样本名  |
| 第2列 | R1路径 |
| 第3列 | R2路径 |
| 第4列 | 分组   |
```

## 运行流程
🔹 Step 1：单细胞分析
```
python up_analysis.py \
  -i sample.xls \
  --work_dir ./01_work \
  --out_dir ./02_result
```
👉 这一步会完成：
1. cellranger分析
2. 表达矩阵生成

🔹 Step 2：RNA editing分析
```
python after_anno.py \
  --work_dir ./01_work \
  --out_dir ./02_result \
  -s sample.xls
```
👉 这一步完成：
1. RNA editing检测
2. 统计分析

## 常见问题
1. 软件未安装
👉 检查：
cellranger
R

2. 路径错误
👉 检查：
```
config.py
```

3. 权限问题
```
chmod -R 755 项目目录
```

4. R包缺失
下载安装缺失R包，例如：
```
conda install r-seurat
```

## 最简运行示例
准备：
1. sample.xls
2. FASTQ数据
3. 软件路径配置完成

执行：
```
python up_analysis.py -i sample.xls
python after_anno.py -s sample.xls
```
